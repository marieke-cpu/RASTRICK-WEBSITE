§ 05 / FAQ

Questions, answered.

How long does a project take?+
A single-page site typically ships in 1–2 weeks. A multi-page build takes 3–4 weeks from kickoff to launch. Timelines depend on how quickly content is supplied and the number of revision rounds we go through.
Do you hand-code every website?+
Yes. Every site I build is written from scratch in HTML, CSS and JavaScript. No page builders, no WordPress, no templates. You get clean, fast, custom code that you actually own.
What technologies do you use?+
HTML, CSS and vanilla JavaScript — the core building blocks of the web. I don't rely on heavy frameworks or bloated libraries. This keeps your site fast, lightweight and easy to maintain long-term.
Do you use WordPress or website builders?+
No. I code everything by hand. Page builders like Wix, Squarespace or WordPress produce bloated code that slows sites down and limits what's possible. Custom code is faster, cleaner and gives you far more control over what your site can do.
Will my website be mobile-friendly?+
Absolutely. Every website I build is mobile-first and tested across devices and screen sizes. A site that doesn't work perfectly on a phone isn't finished — that's not how I ship.
What do you need from me to start?+
A deposit, any existing brand assets (logo, colours, fonts if you have them), content or copy you want on the site, and a clear idea of what you want it to achieve. No brief is too rough — we'll work through it together on a discovery call.
Can I update the content myself after launch?+
Since sites are hand-coded, content updates are handled by me or by editing the files directly. If you need a content management system (CMS), we can scope that in. Either way, I'll hand the site over with clear documentation so nothing is a mystery.
Do you include SEO?+
Yes — SEO foundations are built into every project. This includes semantic HTML structure, meta tags, page speed optimisation, and on-page SEO basics. Ongoing SEO strategy and content optimisation can be scoped separately.
Do you work with businesses outside Australia?+
Yes. I'm based in Canberra but work with clients remotely across Australia and internationally. Everything is handled online — calls, feedback, file delivery and all.
What if I need changes or a redesign of my existing site?+
That's a common starting point. Whether it's a full rebuild, a refresh of an existing design, or adding new pages to what's already live — get in touch and we'll figure out the right scope together.
What happens if I need changes after launch?+
Every package includes a post-launch support window. After that, I offer ongoing care plans for updates, new pages, and performance monitoring. Nothing gets abandoned once it's live.
