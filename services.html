§ 01 / What's included

Three disciplines. One build.

/ 01
Web design

Design isn't decoration — it's strategy made visible. Every layout, every typographic decision, every colour choice is working toward a single outcome: a website your audience feels before they read it. The design phase is where the personality of your business gets translated into something someone else can see and feel. Nothing ships to code until it's right.

  • Visual concept & layout architecture
  • Typography system & colour palette
  • Mobile-first responsive design
  • UI design & component layout
  • Brand-aligned aesthetics
  • Interaction & motion design
  • Asset sourcing & image curation
  • Consistent across all devices & screen sizes
↗
/ 02
Custom development

The design gets built exactly as designed — not approximated. Every site is hand-coded in HTML, CSS and JavaScript. No frameworks adding weight, no builders abstracting away control, no templates underneath. Clean, fast, lightweight code that loads quickly and does exactly what it was built to do. And because I wrote it, I can change anything without untangling someone else's system.

  • Hand-coded HTML, CSS & vanilla JavaScript
  • Zero page builders, zero templates, zero WordPress
  • Custom animations & scroll interactions
  • Contact forms & lead-capture funnels
  • Google Analytics & conversion tracking
  • SEO foundations: semantic HTML, meta tags, speed
  • Cross-browser tested (Chrome, Safari, Firefox, Edge)
  • Clean, documented, maintainable code handover
↗
/ 03
Launch & support

The site doesn't disappear into a ZIP file and that's the end of it. Launch is handled — domain setup, hosting configuration, performance checks, final browser testing. After that, every package includes a post-launch support window for fixes and fine-tuning. Nothing gets abandoned once it's live. Ongoing care plans are available if you need continued updates, new pages, or someone to call when something breaks.

  • Domain & hosting setup and configuration
  • Pre-launch performance & speed optimisation
  • Load speed audit & image optimisation
  • Final cross-device testing before go-live
  • Post-launch revision window (package dependent)
  • Ongoing care plans for updates & new pages
  • Technical support & bug fixes
  • Site health monitoring & uptime

§ 04 / How it works

The process, start to finish.

01

Discovery

A 30-minute call to unpack the business, the goal, and what the site needs to do. No pitch deck — just a real conversation. I listen more than I talk.

02

Scope & proposal

A written proposal within 48 hours — pages, features, package, timeline and fixed price. No vague line items. No hidden extras. No surprises.

03

Design & build

Deposit locks the slot. Design directions land in your inbox first. Once approved, the build begins — hand-coded HTML, CSS and JavaScript from a blank file.

04

Launch & beyond

Final testing, performance check, domain and hosting setup. You get the keys — with full documentation and a support window so nothing is left hanging.
